
# Student Gradebook & Course Manager

## Tina Nouri


## Project Title
Python Final Project - Student Gradebook

## Description
This program manages students, courses, assessments, and grades. 
It allows adding students, creating courses, enrolling students, 
adding assignments, recording grades, and generating reports with averages and letter grades.

## How to Run
Run the program in terminal:
python main.py

## Classes
- Student
- Course
- Assessment
- Quiz
- Exam
- project
- gradebook

## OOP Concepts
- Encapsulation: private attributes in classes
- Inheritance: CourseManager inherits from Gradebook
- Method Overriding: custom print methods

## Creative Features
- Letter grades: convert averages into A, B, C, D, F
- Teacher comment: add a short comment to a student report
